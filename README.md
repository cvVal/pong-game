# Pong

A classic Pong game built with Unity.

## Description

This is a recreation of the classic Pong arcade game featuring two paddles and a ball bouncing between them.

## Getting Started

### Prerequisites

- Unity 6.1
- Git

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd Pong
   ```

2. Open the project in Unity:

   - Launch Unity Hub
   - Click "Open" and select the project folder
   - Unity will import the project automatically

3. Play the game:
   - Click the Play button in Unity Editor

## Controls

- **Player 1**: W/S keys
- **Player 2**: Arrow Up/Down keys

## Built With

- Unity Engine
- C#
